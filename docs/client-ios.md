# iOS 客戶端設定

本文件說明如何在 iPhone / iPad 上安裝與設定 RustDesk 客戶端，連線至自架伺服器。

> **使用前須知**：連線伺服器所需的 **ID 伺服器地址**、**API 伺服器地址** 與 **Key（公鑰）**
> 為實驗室內部資訊，請需要使用的夥伴向**實驗室 IT 管理員**申請取得後再進行設定。    

> Android 的設定流程與本文一致，僅安裝來源不同（Google Play 或官網 APK）。

詳細操作請參考 [RustDesk 官方文件](https://rustdesk.com/docs/zh-tw/client/)。

## 安裝

1. 開啟 **App Store**
2. 搜尋 **RustDesk**
3. 點擊「取得」完成安裝
4. 首次開啟時，授予必要的網路存取權限

---

## 設定自架伺服器

### 步驟

1. 開啟 RustDesk App
2. 點擊右下角 **設定**（齒輪圖示）
3. 點擊 **ID/中繼伺服器**
4. 填入以下資訊（向管理員索取）：

| 欄位 | 值 | 說明 |
|------|---|------|
| **ID 伺服器** | `your-domain.com` | 你的伺服器域名或公網 IP |
| **中繼伺服器** | （留空） | 由 hbbs 自動分配 |
| **API 伺服器** | `your-domain.com` | 你的伺服器域名或公網 IP |
| **Key** | `<id_ed25519.pub 的內容>` | 向管理員索取 |

5. 返回主畫面，下方狀態應顯示 **「就緒」**

> **提示**：Key 為一長串英數字串，建議管理員透過即時通訊傳送，在手機上直接複製貼上，避免手動輸入錯誤。

## 連線遠端設備

### 控制遠端桌面

1. 在主畫面的「遠端桌面」欄位輸入**對方的 ID**
2. 點擊 **連接**
3. 輸入對方的連線密碼
4. 進入遠端桌面

### 觸控操作

| 手勢 | 動作 |
|------|------|
| 單指點擊 | 滑鼠左鍵 |
| 長按 | 滑鼠右鍵 |
| 雙指捏合 | 縮放畫面 |
| 雙指滑動 | 捲動 |
| 三指滑動 | 移動畫面 |

工具列提供虛擬鍵盤、全螢幕切換與連線品質調整。

## iOS 限制與注意事項

### iOS 不能作為被控端

受 Apple 系統限制，iOS 版 RustDesk **只能作為控制端（連出）**，無法接受他人連入控制你的 iPhone / iPad。這是 iOS 平台限制，並非 RustDesk 的問題。

### 背景連線

iOS 會在 App 進入背景後限制網路活動。長時間遠端操作時：

- 保持 RustDesk 在前景
- 避免切換到其他 App，否則連線可能中斷

### 電池消耗

遠端桌面串流會持續使用網路與 GPU 解碼，建議連接充電器使用。

## 常見問題

### 設定 ID Server 後仍顯示「未連線」

1. 確認域名格式正確（不加 `http://` 或 port 號）
2. 確認手機網路正常（切換 Wi-Fi / 行動數據測試）
3. 確認伺服器 21116/udp 已開通

### Key 貼上後格式異常

iOS 複製貼上有時會附帶隱藏字元。確認：

- Key 前後無空格或換行
- 未包含引號
- 建議清除欄位後重新貼上

### 連線畫面模糊或卡頓

1. 工具列 → **顯示** → 調低畫質或降低解析度
2. 使用 Wi-Fi 而非行動數據以獲得更穩定的頻寬
3. 若伺服器啟用了 `ALWAYS_USE_RELAY`，延遲取決於伺服器網路品質
